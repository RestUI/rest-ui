## Installation

You can install the standalone package via **NPM/YARN**. 

### NPM 

```bash
npm install --save rest-ui
```

### YARN
```bash
yarn add rest-ui
```

We also recommnend you using **create-rest-ui-app** to create a rest-ui bootstrapped project!
